<p align="left">Aplicativo Android desenvolvido como avaliação da disciplina de Programação para Dispositivos Móveis. O projeto apresenta funcionalidades relacionadas a recepção e visualização de dados GNSS (Global Navigation Satellite System).</p>

###

<h2 align="left">Funcionalidades</h2>

###

<p align="left">⤷ Tela inicial com navegação para as telas do app.<br><br>⤷ Coleta de localização do dispositivo.<br><br>⤷ Exibição de dados de posição (latitude, longitude, precisão, etc.).<br><br>⤷ Visualização gráfica da esfera celeste com satélites visíveis, sinalizando quais foram usados no FIX.<br><br>⤷ Persistência de preferências do usuário (filtros por constelação, exibir ou não satélites não usados, etc).</p>
